Minimum RAM

2 GB RAM for smooth operation with basic file transfers.

Storage

Network

Wi-Fi or Ethernet connection on same LAN (Local Area Network).

Windows 10+, macOS 10.15+, Ubuntu 20.04+, or any Linux with Node.js.

Browser

Chrome, Edge, Safari, or Firefox for web interface access.
