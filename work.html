How LANDrive Works

LANDrive is a fast, secure peer-to-peer file sharing system that works entirely on your local network. No internet. No cloud. Just direct device-to-device sharing.

8 Simple Steps to Get Started

Step 1: Install LANDrive

Download and install the LANDrive application on your computer using the provided .exe installer.

✔ Works on Windows (Mac/Linux support available)
✔ No internet required after installation

Step 2: Launch the Application

Once launched, LANDrive automatically:

📁 Creates a default shared folder on your system
🌐 Starts a local server (default: http://localhost:7070)
⚡ Initializes real-time communication using Socket.io

Step 3: Connect Devices on Same Network

Ensure all devices are connected to the same WiFi network.

Example:
http://192.168.1.5:7070

Open this URL on any phone, laptop, or tablet in the same network.

Step 4: Access via Browser

LANDrive provides a browser-based interface. No app installation required on other devices.

✔ Works on Chrome, Edge, Safari
✔ Mobile + Desktop compatible
✔ Instant UI loading

Step 5: Start Sharing Files

You can now perform file operations in real-time:

📁 Upload files (Drag & Drop supported)
⬇ Download files instantly
✏ Rename or delete files
📊 Switch between Grid & List views

Step 6: Real-Time Sync

All connected devices stay in sync using Socket.io.

🔄 Changes reflect instantly across devices
⚡ No refresh required
🚀 Zero latency inside LAN

Step 7: Customize Shared Folder

You can change the shared directory anytime from settings.

⚙️ Open Settings → Change Folder
📂 Select any directory on your system

Step 8: Fully Private & Secure

LANDrive operates only within your local network.

🔒 No data leaves your network
🚫 No cloud servers
🛡️ Complete privacy control
